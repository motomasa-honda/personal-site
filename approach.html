どう仕事するかを明示する

一人事業主の信頼性ギャップは、仕事の進め方を透明にすることで埋められると考えています。

01 working principles
01見積は無料、提案書に固定価格を明記します
02納品物にソースコードを全開示します
03NDA・秘密保持は着手前に締結します
04進捗は Slack / Notion で毎日可視化します
05稼働状況(受注可能枠)を本サイトで常時公開します
02 how we work

Fixed-scope パッケージは1週目ヒアリング、2週目実装、3週目納品・引き継ぎの週次ブレイクダウンで進めます。 使用ツールは GitHub / Slack / Notion。特別なツール導入を必要としません。

03 security & confidentiality
✓NDAテンプレート開示 — 着手前にご確認いただけます
✓データ取り扱いポリシーを契約書に明記
✓オンプレ / VPC 構成の技術詳細を事前共有
